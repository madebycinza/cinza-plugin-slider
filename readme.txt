A minimal slider plugin built with Flickity.

== Description ==
Cinza Slider is a minimal slider plugin built with Flickity. It makes lightweight and flexible sliders that can be easily customized with CSS.

== Installation ==
Upload the Cinza Slider plugin to your WordPress site, activate it, and start creating sliders.
